Anira JS

Real-time neural audio processing in the browser

Demos

  • Simple Gain Stereo
    Applies a gain effect to stereo audio using a WASM-based ONNX backend.
  • Streaming Gain Stereo
    Streams audio through a gain effect with real-time processing.
  • JS Callback
    Uses a custom JavaScript callback as the inference backend.
  • ONNX Runtime Web Backend
    Runs inference using ONNX Runtime Web directly in the browser.
  • JS Copying
    Copies input audio directly to output using the JSCopyBackend in JavaScript.
  • PrePostProcessors
    Uses a JSPrePostProcessor for custom JavaScript-based pre/post processing in the inference pipeline.
